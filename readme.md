## 不完全ネジ部


![図1](incompleted_screw.png)
![図2](incompleted_screw2.png)

用語： 不完全ねじ部

定義：
ネジの加工工具の面取り部又は食い付き部などによって作られた山形が不完全なネジ部（付図2102）。
円筒部をもつおねじ部品では、円筒部と完全ねじ部（※1）との境界部及びネジ先端部に、不完全ねじ部が生じる。
全ネジのおねじ部品では、首下部及びねじ先端部に生じる。
